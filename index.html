Git Config — देखो / Check करो

Local
git config user.name git config user.email
Global
git config --global user.name git config --global user.email
✏️

Git Config — Add / Set करो

Local
git config user.name "user-name" git config user.email "email-id"
Global
git config --global user.name "user-name" git config --global user.email "email-id"
🆕

New Repository — Command Line se Banao

echo "# git_tutorial" >> README.md git init git add README.md git commit -m "first commit" git branch -M main git remote add origin https://github.com/user/repo.git git push -u origin main

Existing Repository Push करो

git remote add origin https://github.com/user/repo.git git branch -M main git push -u origin main

  2. उस file में ignore करने वाले file का नाम लिखो, जैसे notes.md

Branch — बनाओ, देखो, Switch करो, Merge करो

बनाओ
git branch <branch-name>
एक साथ बनाओ + Switch करो
git switch -c "new_branch"
सब Branches देखो
git branch * main ← (तुम यहाँ हो, * का मतलब current branch) new_branch
Switch करो
git switch "new_branch" git switch "main"
Merge करो main branch से करो
→ पहले main branch पर जाओ, फिर: git merge "new_branch"
⚠️ Merge Conflict: जब एक ही जगह दो branches ने अलग code लिखा हो तो conflict आता है। VS Code 4 options देता है —
i. Accept Current Change
ii. Accept Incoming Change
iii. Accept Both Changes
iv. Compare Changes
Delete करो main से करो
git branch -d <branch-name>
📦

Stash — Changes याद रखो (Add किए बिना)

Stash करो
git stash
वापस लाओ
git stash apply
साफ करो
git stash clear
🤝

Collaboration का पूरा Process

👑 Main User — Setup
  1. Initial file बनाओ
  2. GitHub पर upload करो (private repo)
  3. Collaborator add करो (username से)
👤 Collaborator — काम करो
  1. Repo clone करो — git clone <link>
  2. पहले नई branch बनाओ — git switch -c "new_branch"
  3. अपनी branch में code लिखो
  4. GitHub पर push करो — git add . git commit -m "message" git push -u origin new_branch
  5. Main user को inform करो कि changes push हो गए
👑 Main User — Merge & Push
  1. Collaborator की branch fetch & merge करो: git fetch git switch "new_branch" git switch main git merge "new_branch"
  2. Merged repo push करो — git push -u origin main
  3. Collaborator latest changes pull करे — git pull origin main
